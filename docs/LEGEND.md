# Legend

Every label scheme used in this repository, in the
[program](https://github.com/arithmon/program) charter, and in the
[atlas](https://github.com/arithmon/atlas) resolves here. If a public Arithmon
document mentions `M3`, `C4`, `N2`, or a `D`-numbered decision, this is the key.

| Label | Meaning | Defined in |
|-------|---------|------------|
| `M1` to `M6` | Milestones of the methodology paper | [`LEDGER.md`](LEDGER.md) |
| `C1` to `C5` | Framework-independent inclusion criteria for the observable list (dimensionless; measured; category completeness; algebraic independence within category; scheme and scale declared) | [`../freeze/OBSERVABLES_FREEZE.md`](../freeze/OBSERVABLES_FREEZE.md) |
| `Cat A` to `Cat G` | The admitted observable categories (gauge couplings, charged-lepton ratios, quark ratios, CKM, PMNS, boson ratios, cosmology) | [`../freeze/OBSERVABLES_FREEZE.md`](../freeze/OBSERVABLES_FREEZE.md) |
| `N0` to `N3` | The four null models: accidental-match baseline (N0), complexity-matched expression search (N1), adversarial alphabet (N2), permutation (N3) | [`../scripts/`](../scripts/) and [`LEDGER.md`](LEDGER.md) |
| `G_INT`, `G_STRUCT`, `G_TRANS` | The three expression grammars: small-integer arithmetic baseline, declared structural alphabet, transcendental extension | [`../freeze/GRAMMARS_FREEZE.md`](../freeze/GRAMMARS_FREEZE.md) |
| `kappa_MDL`, `kappa_NODE` | The two complexity measures: description length in bits (primary), bare node count (secondary) | [`../freeze/GRAMMARS_FREEZE.md`](../freeze/GRAMMARS_FREEZE.md) |
| `D1`, `D2`, ... | Dated entries in the decisions ledger | [`LEDGER.md`](LEDGER.md) |

A worked case study assigns its own per-relation claim identifiers (for example
an `A`-prefixed list for a scored framework). Those are defined where that case
study is published, not here: this repository is the instrument, and a worked
case study lands in its own paper.
